### Resumen

Este proyecto tiene como objetivo implementar, en Python, una canalización completa de datos que tome información desde una fuente externa, la almacene y la procese para poder analizarla. A lo largo del trabajo se realizarán análisis exploratorios con estadística descriptiva, se construirán y ajustarán modelos de probabilidad junto con sus parámetros y se representarán tanto los datos como sus distribuciones de forma gráfica. Para lograrlo, se utilizarán APIs para obtener datos de manera programática,  y se trabajará con bases de datos como SQLite  empleando SQLAlchemy para mapear los datos a objetos. Con todo ello, se busca no solo extraer y organizar la información, sino también aplicar herramientas estadísticas de Python para obtener conclusiones sobre los datos analizados.
